# 포트폴리오 설명 문구

## 이력서 한 줄 요약

Google News RSS 기반 뉴스 기사 크롤링 및 감정 분석 파이프라인 구축

## 상세 설명

Google News RSS를 활용하여 특정 키워드의 뉴스 기사 목록을 수집하고, 기사 제목·요약·본문 후보 텍스트를 기반으로 감정 분석을 수행하는 데이터 수집 및 분석 프로젝트를 진행했습니다.

단순히 제공된 CSV를 분석하는 것이 아니라, RSS 요청을 통해 원본 데이터를 직접 수집하고, 수집된 텍스트를 전처리한 뒤 사전 기반 감정 점수를 계산하여 긍정·중립·부정으로 분류했습니다. 이후 감정 분포와 주요 키워드 빈도를 분석하고 CSV 및 그래프 형태로 결과를 저장하는 파이프라인을 구현했습니다.

## 기술 스택

- Python
- requests
- BeautifulSoup
- feedparser
- pandas
- matplotlib
- Web Crawling
- Text Preprocessing
- Sentiment Analysis

## 면접 답변용 문구

이 프로젝트는 뉴스 데이터를 직접 수집하고 분석까지 이어지는 과정을 경험하기 위해 진행했습니다. 일반 뉴스 웹페이지는 사이트별 구조가 다르고 접근 제한이 있을 수 있기 때문에, 안정적으로 기사 목록을 가져올 수 있는 Google News RSS를 활용했습니다. 이후 기사 링크에 접근해 본문 수집을 시도하고, 제목·요약·본문 텍스트를 통합하여 감정 분석을 수행했습니다.

다만 사이트별 본문 구조가 모두 다르기 때문에 본문 수집률에는 한계가 있었고, 이 부분은 실무에서도 크롤링 설계 시 반드시 고려해야 하는 점이라고 판단했습니다. 이후 개선 방향으로는 Selenium 기반 동적 페이지 처리, 더 정교한 한국어 감정 사전, KoBERT 기반 감정 분류 모델 적용을 고려했습니다.
